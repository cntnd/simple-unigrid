Simple Code

All the code you need is simple and familiar. Simply state the number of columns you want your content to occupy in the .brick class. For example, if you want your content to take up 8 columns (out of 12), simply give your content the class .brick--8.

Simple grid is built mobile-first, so all columns will expand to the full container width on smaller screens. If you donâ€™t want columns to expand on mobile devices and small screens, simply use .brick-sm as your column class name. For example, if you want to have two blocks of content floating side-by-side on small screens, each would be given the class name .brick-sm--6.

Be sure to nest columns within a .unigrid-row class. You may also choose to nest rows within a .container class.

        
<div class="unigrid-row">
    <div class="brick--4">
        <p>brick--4</p>
    </div>

    <div class="brick--4">
        <p>brick--4</p>
    </div>

    <div class="brick--4">
        <p>brick--4</p>
    </div>
</div>
